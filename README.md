# ZetaMacPlus

A modern upgrade to the popular mental math trainer arithmetic zetamac. 

🔗 https://zetamacplus.com


## Features

- 4 unique game modes, auto-refreshed daily level, custom mode.
- User Stats tracking on a per-problem basis.
- Post runs solve-speed and problem solve time analytics.
- Dynamic global leaderbaord
- Live 1v1 versus gamemode.

## Tech stack
- **Frontend/ Backend:** NextJS
- **Database:** PostgreSQL
- **Auth:** BetterAuth
- **Deployment:** Vercel

### Installation

\`\`\`bash
git clone https://github.com/yourusername/zetamacplus.git
cd zetamacplus
npm install
npm run dev
\`\`\`

App will run at `http://localhost:3000`.
